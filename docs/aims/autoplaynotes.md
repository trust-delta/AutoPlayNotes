# IS
好きな曲を弾きたいのに、**譜面も、技術も、音楽の知識も無い**。この 3 つのギャップを、素材が何であれ埋めきる、という読み。

3 つは独立した穴ではない。**「演奏範囲（窓）」が全部を貫く。** だから窓だけは子 aim へ配らず、ここに置く。

- 窓は**技術**のギャップを埋める道具である（あなたが押すキーを減らす）→ [[playable-reduction]]
- 窓が空のときの縮退形が**自動演奏**＝入口である → [[practice-onboarding]]
- 窓の外を鳴らす `merge()` が、**耳での検証**を可能にする → [[ear-verification]]
- 窓に入れる譜面そのものを、どんな素材からでも得る → [[score-intake]]
- ビルドできない人へ届ける → [[shipping]]

窓 ＝ ユーザーが担当する連続したキー範囲。窓の外はアプリが引き受け、その引き受け方と直積を取ると、自動演奏から練習までが 1 本の連続した面の上に並ぶ。

| あなたの窓 | 窓の外をアプリが | モード | ゲームへのキー送出 |
|---|---|---|---|
| 空 | ゲームへ送出 | 自動演奏（**入口**） | あり |
| 1 窓〜 | ゲームへ送出 | 補助演奏 | あり |
| 空 | スピーカー | 単純再生（譜面確認） | なし |
| 1 窓〜 | スピーカー | 練習＋伴奏 | なし |
| 1 窓〜 | 鳴らさない | 練習 | なし |

- **窓で分ける必然**（実測）: あなたのキー集合とアプリのキー集合が全曲を通じて素になり、物理キーの取り合いが構造的に消える。同時押し数で分けると役割が時間をまたいで入れ替わり、アプリの keyup があなたの音を切る（OS のキー状態は 1 つしかない）。補助演奏が安全に成立する根拠。
- **音長は演奏の質そのもの**: 対象楽器は「押している間だけ鳴り、押下エッジでしか発音しない」。`NoteEvent.durations`（音ごとの音長）・同キー衝突の後勝ち解決・`win_input.high_resolution_timer`（`timeBeginPeriod(1)`）で押下時間の誤差 ±0.8ms。この前提は 3 つのギャップすべてに効くので root に置く（→ ESCALATION）。
- **自動演奏エンジン**: `keymap` の音階→キー割当を通し、`win_input`（スキャンコード SendInput）＋ `player`（別スレッド・停止可・和音・ヒューマナイズ）でフォーカス窓へキーを送る。
- **楽器はピアノに限定する。** 多様な楽器パターンの網羅は非現実的。`KeyMapping.sustain` は残す（打楽器プリセット用・実装済みでコストゼロ）。
- **周辺**: `staff`（五線譜プレビュー / 簡易エディタ）・`playlist`・`theme` / `config`・`gui`（customtkinter）。v0.1.0 リリース済み。

# ESCALATION
**設計の土台が実機で未検証。Go だけでは進めない。**

- 「押している間だけ鳴り、押下エッジでしか発音しない」という楽器の挙動は、**出典がユーザーの知識であって実測ではない**。
- プリセットが前提する鍵盤数も未確認。`chromatic` の低音の記号キーは JIS 配列で物理位置が違いうる。窓の既定値がここに依存している。
- **ゲームを動かせる環境が要る。** ここが倒れると、音長・補助演奏・窓の既定値の前提が同時に変わる。
- 仮に「押している間鳴る」が偽（＝撥弦楽器）なら `KeyMapping.sustain=False` で従来動作に戻せる。「押下エッジでしか発音しない」が偽なら再発音時の up が不要になる（害は無い）。

# HISTORY
- **旧 IS「最も広い入力 → 変換 → 2 系統の出力（自動演奏 / 練習）を対等に並べる」**（2026-07-09 棄却）。自動演奏で来る人は練習したい人と別のセグメントではなく、あきらめた同一人物である。対等な 2 出力として置くと**両者をつなぐ機構が設計に現れず**、実際 `Level` に速度と自動演奏フラグを混ぜる誤りを生んだ。自動演奏は出力ではなく、窓が空のときの縮退形。
- **旧前提「ゲーム内楽器はオンセットだけの世界だから音長は要らない」**（棄却）。`midi_parser` が `note_off` を捨て、`player` が `gate=40ms` を上限にして全曲スタッカートにしていた。3 層すべてが同じ誤った前提の上に建っていた。テストが 1 本も無い層にバグが住んでいた。
